# ESP32-P4 M5Tab SSH 客戶端快速使用手冊

[English](QUICK_START_GUIDE.md) | [简体中文](QUICK_START_GUIDE_zh-CN.md) | **繁體中文**

本指南只保留日常使用需要知道的內容。繁體中文版本會重點說明終端裡的實體鍵盤、軟鍵盤和拼音輸入法。

## 主要特性

1. **SSH 終端客戶端**：面向 M5Stack Tab5 的觸控螢幕 SSH 客戶端，支援保存多個伺服器，短按即可快速連接。
2. **多會話保持**：伺服器列表會保留會話設定，斷開後可以回到主畫面繼續切換或重新進入常用伺服器。
3. **中文拼音輸入法**：終端內建拼音輸入，支援觸控軟鍵盤和實體鍵盤輸入中文。
4. **實體鍵盤增強**：偵測到實體鍵盤後自動隱藏軟鍵盤，支援方向鍵、Tab、Esc、Ctrl 組合鍵等常用終端操作。
5. **TF 卡檔案管理器**：可以在設備上瀏覽、打開、複製、剪切、貼上、重新命名、刪除 TF 卡檔案；實體鍵盤也可以參與檔案管理器操作。
6. **線上檔案管理器**：連接 WiFi 後，可在設定中開啟線上 TF 卡檔案管理器，透過瀏覽器存取設備檔案。
7. **OTA 韌體升級**：設定頁可偵測主程式和 UPLOAD 韌體更新，下載到 TF 卡後按提示完成升級。
8. **網路與電源狀態**：主畫面顯示 WiFi、IPv4/IPv6、電量、USB-C 供電和充電狀態，方便隨時確認設備狀態。

## 1. 首次啟動

1. 將 `flash-at-0x0/` 中的完整韌體刷入 `0x0` 位址。
2. 如需檔案管理、截圖、音樂播放、OTA 包儲存或設定備份，請插入 TF 卡。
3. 開機後選擇 **繁體中文**。
4. 進入 SSH 伺服器列表。

後續可在 **系統設定 > 系統 > 語言** 中切換語言。切換語言後設備會顯示「切換中」，隨後關閉螢幕並重啟。

## 2. 主畫面

啟動後進入「SSH 伺服器」列表。

主畫面常用入口：

1. **伺服器卡片**：短按連接；長按打開「編輯 / 刪除 / 取消」選單。
2. **新增**：右上角「+ 新增」，新增 SSH 伺服器。
3. **WiFi 狀態**：右上角 WiFi 晶片，進入 WiFi 管理。
4. **電源狀態**：頂部電量/USB-C 區域，查看電源資訊和充電速度。
5. **IPv4 / IPv6**：底部顯示目前網路位址狀態。
6. **設定**：底部設定入口，用於應用設定。

新增伺服器時需要填寫：圖示、名稱、主機/IP、連接埠、使用者名稱、密碼，以及可選的「連接後自動執行命令」。

## 3. WiFi 管理

進入 WiFi 管理後可以：

1. 查看目前連接狀態、IPv4/IPv6 位址。
2. 手動輸入 SSID 和密碼，點擊「保存並連接」。
3. 在「WiFi 列表」中選擇掃描到的網路。
4. 對已保存網路執行「連接」或「刪除」。

開放網路可以不填密碼。掃描列表裡，「直連」表示開放網路可直接連接，「使用」表示先填入 SSID，再輸入密碼連接。

## 4. 進入終端

短按伺服器卡片後進入 SSH 終端。終端畫面預設盡量全螢幕，只在需要時顯示控制區。

觸控熱區：

1. **頂部邊緣**：顯示狀態列，狀態列右側「X」可斷開連接並返回伺服器列表。
2. **左側邊緣**：顯示/隱藏控制列。
3. **右側邊緣**：顯示/隱藏軟鍵盤和控制列。

控制列包含：輸入法切換、ESC、TAB、CTRL、ALT、HOME、END、PgUp、PgDn、方向鍵、Enter。

## 5. 終端輸入法和鍵盤操作

繁體中文系統預設使用中文輸入優先。拼音模式下，先輸入拼音，選擇候選字後才會把漢字發送到 SSH 終端；英文模式下，字母會直接發送到終端。

### 5.1 輸入模式

1. **拼音模式**：輸入 `zhong` 這類拼音後，螢幕會顯示拼音輸入列和候選字列。
2. **英文模式**：輸入字母、數字、符號會直接發送到遠端 shell。
3. **切換方式**：
   - 軟鍵盤/觸控：打開左側或右側控制列，點「拼 / EN」按鈕。
   - 實體鍵盤：按 `Ctrl + 空格`。

切換時螢幕會短暫顯示「拼」或「EN」的提示圖示。

### 5.2 軟鍵盤輸入

沒有偵測到實體鍵盤時，點擊右側熱區可以彈出軟鍵盤。

| 操作 | 說明 |
|---|---|
| 輸入拼音字母 | 在拼音模式下組成拼音，並顯示候選字 |
| 點候選字 | 直接把該漢字發送到終端 |
| 空格 | 選擇第 1 個候選字 |
| 方向鍵左/上 | 候選字上一頁 |
| 方向鍵右/下 | 候選字下一頁 |
| 回退 | 有拼音時刪除一個拼音字母；無拼音時發送退格 |
| Enter/確認 | 有拼音未選字時，發送目前拼音字母；無拼音時發送回車 |
| 關閉鍵盤 | 隱藏軟鍵盤，並清除未完成的拼音輸入 |

如果要輸入英文命令，建議先切到 **EN**，例如輸入 `ls -la`、`cd /tmp` 這類命令會更直接。

### 5.3 實體鍵盤輸入

偵測到 Tab5 實體鍵盤後，終端會自動隱藏軟鍵盤，直接使用實體鍵盤輸入。

| 按鍵 | 拼音模式 | 英文模式/普通終端 |
|---|---|---|
| 字母 | 組成拼音 | 直接發送字母 |
| 數字 `1-9` | 選擇對應候選字 | 直接發送數字 |
| 數字 `0` | 選擇第 10 個候選字 | 直接發送數字 0 |
| 空格 | 選擇第 1 個候選字 | 發送空格 |
| 方向鍵左/上 | 候選字上一頁 | 發送方向鍵 |
| 方向鍵右/下 | 候選字下一頁 | 發送方向鍵 |
| Backspace | 刪除一個拼音字母 | 發送 `DEL (0x7F)` |
| Enter | 發送目前拼音字母 | 發送回車 |
| Esc | 清除目前拼音輸入 | 發送 ESC |
| Tab | 發送 Tab | 發送 Tab |
| Ctrl + 空格 | 切換「拼音 / 英文」 | 切換「拼音 / 英文」 |
| Ctrl + 字母 | 發送控制字符，如 `Ctrl+C` | 發送控制字符 |
| Alt + 字符 | 發送 `ESC + 字符` 序列 | 發送 `ESC + 字符` 序列 |

注意：拼音模式下，如果已經輸入了拼音，數字和空格會優先用於選字；想直接輸入數字或空格，可以先選完字，或切到 **EN**。

## 6. TF 卡檔案管理器

檔案管理器支援：

1. 瀏覽 TF 卡目錄和檔案。
2. 使用頂部工具列執行全選、複製、剪切、貼上、刪除。
3. 刪除前會彈出確認對話框。
4. 打開支援的圖片、文字和音樂檔案。
5. WiFi 環境下開啟線上檔案管理器，透過瀏覽器上傳或下載檔案。

截圖統一保存到 TF 卡 `/ScreenShots`。

## 7. 音樂播放器

檔案管理器可打開 MP3/FLAC 音樂。

支援：

1. 從目前資料夾或收藏夾播放。
2. 在音樂列表中加入或移出收藏夾。
3. 切出音樂介面後繼續後台播放。
4. 讀取歌曲內嵌封面並顯示。

音樂播放和圖片查看都比較佔資源。如果需要做大量檔案操作或看大圖，建議先暫停音樂。

## 8. OTA 韌體升級

進入 **系統設定 > 韌體升級**。

可以：

1. 偵測主程式更新。
2. 偵測 UPLOAD 韌體更新。
3. 下載韌體包到 TF 卡。
4. 版本和檔案校驗通過後按提示安裝。

升級時請保持供電穩定，不要拔 TF 卡或斷電。

## 9. 設定備份

進入 **系統設定 > 設定備份**。

| 類型 | 用途 |
|---|---|
| 加密備份 | 推薦用於跨設備遷移，SSH/WiFi 密碼受備份密碼保護 |
| 非加密備份 | 適合本機恢復，請注意隱私風險 |

SSH 和 WiFi 憑據屬於敏感資訊，請妥善保管備份檔案。

## 10. 快捷速查

| 目標 | 觸控/軟鍵盤 | 實體鍵盤 |
|---|---|---|
| 顯示狀態列 | 點頂部邊緣 | - |
| 顯示控制列 | 點左側邊緣 | - |
| 顯示/隱藏軟鍵盤 | 點右側邊緣 | 偵測到實體鍵盤後自動隱藏 |
| 斷開 SSH | 狀態列右側「X」 | - |
| 切換拼音/英文 | 控制列「拼 / EN」 | `Ctrl + 空格` |
| 發送 `Ctrl+C` | 控制列 `CTRL` 後按 `C` | `Ctrl + C` |
| 發送 ESC | 控制列 `ESC` | `Esc` |
| 發送 Tab | 控制列 `TAB` | `Tab` |
| 候選字翻頁 | 控制列方向鍵 | 方向鍵 |
| 選第 1 個候選字 | 空格或點候選字 | 空格 |
| 選第 N 個候選字 | 點候選字 | 數字鍵 |
